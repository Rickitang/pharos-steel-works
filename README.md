# Pharos Steel Works

Professional metal fabrication services website for Pharos Steel Works, located in Kempton Park, South Africa.

## Website Features

- **Services**: Sheet laser cutting, pipe/tube cutting, sheet bending, steel rolling, CNC fiber welding, powder coating
- **Responsive Design**: Mobile-first approach with full tablet and desktop support
- **SEO Optimized**: Structured data, sitemap, meta tags
- **Accessibility**: ARIA labels, skip links, keyboard navigation
- **Performance**: Lazy loading, debounced events, optimized images
- **Security**: CSP headers, HTTPS enforcement, form protection

## Technology Stack

- HTML5
- CSS3 with CSS Variables
- Vanilla JavaScript (ES6+)
- FormSubmit.co for contact form handling

## Deployment

This is a static website ready for deployment to any hosting platform.

### Netlify Deployment

1. Push to GitHub
2. Connect repository to Netlify
3. Deploy settings:
   - Build command: (none required)
   - Publish directory: `.` (root)
4. Configure custom domain: pharossteelworks.co.za
